# 主題

隨著網際網路的發展，圖書館的功能已可以經由在網際網路上的網頁管理，藉由資料庫與網頁的結合，可以完成以往管理圖書的繁瑣過程。我們將利用前端與後端的技術呈現圖書館的管理系統。

藉由本次的專案，可以練習到：

1. php
2. php 模板引擎
3. 物件導向
4. 資料庫的 CRUD（新增、讀取、更新與刪除）
